# Proyecto_TC1033.303

## Contexto
Un pequeño invernadero necesita un programa el cual muestre informacion, caracteristicas y cuidados de plantas que tienen aa su disposicion. Asimismo, necesitan saber los precios de las plantas por orden de compra. El programa muestra todas las caracteristicas solicitadas de cada planta.

## Funcionalidad:
El programa imprime la informacion de la planta deseada y muestra los precios por si el usuario desea comprar alguna. 
El programa permite: 
- Elegir el tipo de planta del cual necesita la información
- de ese tipo, se muestra qué informacion deseas saber
- se puede hacer pedido de plantas para comprar y despliega el precio
