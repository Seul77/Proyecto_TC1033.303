# Proyecto_TC1033.303

## Contexto
En un invernadero se manejan diferentes tipos de plantas, cada una con características específicas. Se desea un programa que le permita tener distintos tipos de plantas, para así organizar mejor la información y entender las necesidades particulares de cada una.
El programa debe permitir:

- Crear diferentes tipos de plantas.

- Mostrar la información principal de cada una.


## Funcionalidad:
Por ahora, el programa despliegara las caracteristicas de cada una de las plantas y se pueda tener a facil disposicion la informacion de cada una de estas.
